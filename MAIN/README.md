<h1> Algumas Obervações </h1>
<li>Configure um ambiente virtual para o projeto.</li>
<li> Após a instalação de biblioteca, e seu efetivo uso, deverá atualiza o arquivo <b> <u>requirements.txt </u></b> na raiz do projeto.  <code>pip freeze > requirements.txt </code>.</li>
<li> Toda a aplicação deverá conter um <b>README.md</b>.</li>


<h2> Todo o projeto deverá se instanciado na aba Projetos do <b>GitHUB</b> pievdcs</h2>
<span>
    Aonde deverá se aberto as 'Issues' e posterior as 'Branch',
    O projeto não poderá ser desenvolvido em sua 'MAIN', mas em suas <i>Branch</i>, facilitando a manutenção do código e permitindo a confecção do mesmo em equipes.

</span>
